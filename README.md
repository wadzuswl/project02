## 📌 프로젝트 소개
카페24 템플릿을 참고하여 제작한 **반려동물 분양·입양 서비스 랜딩페이지 클론 코딩 프로젝트**입니다.  
UI 구조와 레이아웃을 분석한 뒤, HTML/CSS/JavaScript만을 사용해 직접 구현했으며  
실제 서비스 페이지처럼 보이도록 섹션 구성과 사용자 흐름에 집중했습니다.

본 프로젝트는 **프론트엔드 포트폴리오용**으로 제작되었습니다.

---

## 🎯 프로젝트 목표
- 실무형 랜딩페이지 레이아웃 구조 이해
- 섹션 단위 UI 설계 및 마크업 연습
- JavaScript를 활용한 인터랙션 구현
- 반응형 웹 페이지 제작 경험

---

## 🖥 주요 섹션 구성
- **Hero 영역**  
  브랜드 메시지를 강조하는 메인 비주얼
- **About**  
  안심 케어 센터 소개 및 주요 지표 정보
- **Services**  
  메디컬 센터 / 트레이닝 센터 서비스 안내
- **Process**  
  상담 → 방문 → 검진 및 서류 → 계약 → 사후관리 단계 구성
- **Adoption**  
  입양 대기 중인 반려동물 카드 UI
- **Reviews**  
  사용자 후기 섹션
- **Branches**  
  지점 정보 및 연락처 안내

- Swiper.js를 활용한 슬라이드 구현
- 모바일 / 데스크탑 반응형 대응

## 📝 회고
- 카페24 템플릿을 분석하며 **실무형 랜딩페이지 구조**를 이해할 수 있었습니다.
- 섹션 단위로 레이아웃을 나누어 작업하면서 **HTML 시맨틱 구조의 중요성**을 체감했습니다.
- SCSS로 스타일을 관리하며 반복되는 UI를 구조적으로 정리하는 연습이 되었습니다.
- JavaScript로 메뉴 토글, 슬라이드 등 UI 상태를 제어하며 **DOM 조작에 대한 이해**가 높아졌습니다.
- 반응형 구현 과정에서 해상도별 여백과 정렬 차이가 UX에 큰 영향을 준다는 것을 배웠습니다.
